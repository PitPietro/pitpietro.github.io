Cryptography

Vocabulary

Vocabulary is is fundamental to understand everything without creating any misunderstanding. Let’s start whit the names of the sciences involved in cryptography:

ASCII
American Standard Code for Information Interchange, a standard that turns alphabetic (and others) characters into numbers.
Asymmetric key cryptography
A kind of encryption where the key required to encrypt is different from the once required to decrypt. Those two keys are mathematically interconnected.
Cipher
A general system that hides the meaning of a message by replacing the letters of the original text (known as plaintext) with other letters.
Cipher Alphabet
The reorganization of the plain (ordinary) alphabet. It determinates how each letter is enciphered.
Ciphertext
The plaintext after after encipherment.
Code
A system that hides the meaning of a message my replacing each word (or phrase) in the original message with another character (or set of characters). The list of replacements in enclose in a codebook.
Cryptanalysis
The science of extrapolate the plaintext from a ciphertext, without knowing the key.
Cryptography
The science of encrypting a message or hide its meaning. The term is usually used to mean of anything connected with ciphers, so it can be a synonym of cryptology.
Cryptology
The science of secret writing. It covers cryptography and cryptanalysis.
Decipher
The action to turn an enciphered message back to a plaintext message.
Decode
The action to turn an encoded message back to a plaintext message.
Decrypt
The action to decipher of to decode.
DES
Data Encryption Standard developed by IBM in 1976. DES is insecure due to the relatively short 56-bit key size.
Digital Signature
A method that provides the authorship of an electronic document.
Encipher
The action to turn the plaintext into the enciphered message.
Encode
The action to turn the plaintext into the encoded message.
Encrypt
The action to encipher or encode.
Encryption Algorithm
Any general encryption process that is performed by choosing a specific key.
Homophonic substitution cipher
A cipher where there are several potential substitutions for each plaintext letter.
Key
The element witch turns a general encryption algorithm into a specific method for encryption.
Key escrow
A scheme in which users lodge copies of their secret key with a trusted third party, who will pass on keys to law enforcers only under certain circumstances.
Key length
Since computer encryption involves keys that are numbers, key length is the number of digits (or bits) in the key. It indicates the number of possible keys.
Monoalphabetic substitution cipher
The cipher alphabet is fixed throughout encryption.
Onetime pad
The unbreakable form of encryption. The key has the same length of the message. Each key can be used only once.
Plaintext
The original message before encrypting.
Polyalphabetic substitution cipher
A substitution cipher in which the cipher alphabet changes during the encryption (like the Vigenère cipher). The change is defined by a key.
PGP
Pretty Good Privacy is a computer algorithm developed by Phil Zimmermann and based on RSA.
Private key
In a system of public key encryption, the receiver use the private key to decrypt messages. It must be kept secret.
Public key
In a system of public key encryption, the sender use the public key to encrypt messages. It must be available to everyone.
Public key cryptography
It overcomes the problems relied on key distribution. It requires an asymmetric cipher: public encryption key and private decryption key.
Quantum cryptography
An unbreakable form of cryptography that exploits quantum theory, especially the uncertainty principle: the impossibility to measure all aspects of an object with absolute certainty. Quantum cryptography guarantees the secure exchange of a random series of bits, which is then used as the basis for a onetime pad cipher.
RSA
The first system that is adapted to suit Public key cryptography requirements. It was developed in 1977 by Ron Rivest, Adi Shamir and Leonard Adleman.
Steganography
The science of hiding the existence of a message.
Substitution cipher
A system of encryption where each letter of a message is replaced with another character. All the characters retain their position within the message.
Symmetric key cryptography
A form of cryptography where the key is required for encrypting is the same as the key required for decrypting.
Transposition cipher
A system of encryption where each each letter of a message changes its position within the message. All the characters retain their identity within the message.
